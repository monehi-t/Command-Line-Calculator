# Command-Line-Calculator

## Project Description
This project is a simple command-line calculator developed in Python.  
It allows the user to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

The purpose of this project is to demonstrate the use of GitHub for version control and project management.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Basic error handling

## Technologies Used
- Python 3

## Requirements
To run this project you need:

- Python 3 installed

## How to Run the Project

1. Download or clone the repository 
2. Navigate to the project folder
3. Run the program
